# Ban Hung Ca Khang Chien 1945-1954

Website lich su ve cuoc khang chien chong thuc dan Phap (1945-1954).

## Cai dat

```bash
npm install
npm run dev
```

## Cong nghe
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Google Fonts (Playfair Display, Crimson Text, Inter)
